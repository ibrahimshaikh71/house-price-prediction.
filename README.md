This project predicts house prices using machine learning techniques. It uses a housing dataset containing features such as area, number of bedrooms, bathrooms, parking spaces, furnishing status, and other property details. The data is preprocessed by encoding categorical variables, followed by exploratory data analysis (EDA) to understand relationships between features. Multiple regression models, including Linear Regression and Random Forest Regressor, are trained and evaluated using metrics such as MAE, RMSE, and R² score. The best-performing model is saved and used to predict the price of a new house based on user-provided property details.

Features
Data loading and preprocessing
Exploratory Data Analysis (EDA)
Categorical data encoding
Linear Regression model
Random Forest Regressor model
Model performance evaluation
Feature importance visualization
Model saving using Joblib
House price prediction for new data
